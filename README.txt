1. What is the purpose of this project? 
  Sometimes great ideas appear in a sudden unexpected time, can happen when: 
  - Sudden daydreaming
  - Pondering upon sth
  It is better to take note on that idea and figure out how to realise it.
2. Who is the project for? For creative people who got sudden idea
need to record it
3. Why is it different?
- Designed specifically to take a note of sudden ideas.
- Provides list of milestone for user to list out the step
to do that ideas
4. How do I use it? (installation instructions)
download the project then open it using Android Studio Developer
